# UniProject
Bibliotečki informacioni sistem - aplikacija za evidenciju rada biblioteke

## Opis projekta
Projekat ce biti aplikacija za evidenciju rada biblioteke. Sistem omogucava vodjenje evidencije o knjigama, autorima, kategorijama, kao i clanovima biblioteke i zaduzenjima knjiga.
Sistem se pravi za koriscenje bibliotekaru. Pod clanovima biblioteke podrazumevaju se osobe koje bibliotekar evidentira u sistem.

Aplikacija ne zahteva registraciju i login korisnika ali radi boljeg razvoja i skupljanja iskustva implementiracu Registraciju i Login za bibliotekare.

Kod i nazive u kodu cu pisati na Engleskom, radi boljeg vezbanja i skupljanja iskustva. Frontend UI cu pisati na Srpskom radi bolje prezentacije na ispitu.

## Struktura
UniProject je root repo.
- Backend, sadrzace Spring Boot (Biznis logika, MySQL konekcija i REST API, Auth JWT)
- Frontend, sadrzace Angular (TypeScript)
